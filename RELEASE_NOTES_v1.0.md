Vorliq Version 1.0 Release Notes
================================

Vorliq Version 1.0 is the first complete release of the Vorliq ecosystem. Vorliq is experimental open-source community blockchain software built on its own lightweight blockchain. The native coin is VLQ, and the network does not depend on Ethereum, Bitcoin, Solana, or any other outside blockchain.

This release includes the full Python proof of work blockchain core with signed transactions, SECP256K1 wallets, persistent chain storage, mining, validation, and token economics. VLQ has a maximum supply of 21 million coins, a starting mining reward of 50 VLQ per block, and a scheduled halving every 210000 blocks. VLQ has no guaranteed market value and should not be treated as an investment promise.

Version 1.0 includes the React web application, the Node.js backend API, and the React Native Expo mobile application. Users can create wallets, send VLQ, mine blocks, inspect the blockchain, view balances, check account history, export transactions, use dark and light mode, receive notifications, and monitor local node health.

The release also includes community lending-style features, where members request VLQ support and other members vote with VLQ balance as voting weight. It includes the decentralized VLQ exchange offer board, where members post buy and sell offers using prices described in plain language. It includes community governance, where VLQ holders create proposals, vote on rule changes, and approved proposals can automatically update supported network settings.

Vorliq is not a licensed bank, broker, exchange, lender, investment adviser, custodian, or financial institution. Mining rewards, treasury rewards, tips, exchange offers, lending activity, price signals, and governance votes are experimental software features and may change over time. Users are responsible for their own keys, local laws, counterparties, posts, offers, votes, taxes, and risk decisions.

Vorliq Version 1.0 includes peer to peer networking, peer discovery, automatic chain synchronization, a public node registry, network diagnostics, persistent storage, rotating logs, backend error handling, a setup guide, a production deployment guide, GitHub Pages documentation, a full Python and backend test suite, GitHub Actions CI, and a multi node stress testing system.

To get started, visit https://vorliq.github.io/Vorliq, read the setup guide, install Git, Node.js LTS, and Python 3.12, clone the repository, install dependencies, and run `start.bat` on Windows. The web application runs at `http://localhost:3000`. The mobile app can be run with Expo Go by opening the `mobile` folder and running `npx expo start`.
