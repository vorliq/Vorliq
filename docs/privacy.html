Privacy Policy

Vorliq is designed to keep wallet keys under the user's control. Vorliq servers do not store user private keys, wallet passwords, encrypted wallet backup passwords, or recovery phrases. If a user loses a private key or wallet password, Vorliq cannot recover it.

Blockchain Data

The server stores public blockchain and community data needed to run the network. This includes blocks, pending and confirmed transactions, public wallet addresses involved in transactions, forum posts and replies, lending requests, exchange offers, governance proposals and votes, treasury proposals, price signals, achievements, public node registry entries, diagnostics, reports, and operational logs.

Transactions, blocks, forum posts, lending proposals, exchange offers, governance votes, treasury proposals, and public node registry entries are public community data. Users should not post private information in public blockchain or community features.

Local Wallet Storage

The web app may store encrypted wallet data in the user's browser localStorage. That local record contains the public wallet address, public key, encrypted private key, encryption salt, initialization vector, and encryption settings. The password used to decrypt it is not stored by Vorliq. The mobile app stores wallet data locally on the user's device.

Users should be careful on shared, borrowed, public, or work-managed devices. Anyone with access to the device, browser profile, device backups, malware, or screen recording tools may be able to attack wallet data or see sensitive information if the user reveals a private key.

Newsletter and Forms

Newsletter and ambassador forms are handled through Formspree. When a user submits an email address or application, Formspree processes that submission and forwards it to Vorliq. Users should review Formspree's privacy practices before submitting personal information.
