Vorliq

Your Community Your Coin

Vorliq is a self contained community savings platform built on its own blockchain. It lets people create wallets, send VLQ, mine blocks, vote on community lending, and keep a shared financial record without depending on an outside cryptocurrency network.

Features

Own Blockchain

Vorliq runs on its own lightweight proof of work blockchain. Blocks, transactions, wallets, balances, lending records, and mining all belong to the Vorliq application itself.

Real Cryptography

VLQ transactions use real SECP256K1 cryptographic keys and signatures. The chain verifies signed transactions before they can be mined into blocks.

Community Governed

Vorliq is built for community ownership. Members can participate in savings, lending votes, node discovery, and future governance through VLQ participation.

Live Network

Vorliq is currently running as a community network that people can download, run, and connect through peer discovery. Each member can operate a local node, register it with the public node registry, and help strengthen the shared VLQ network.

To join, download the application from GitHub, follow the README setup instructions, start the app, create your wallet, and connect with other community nodes through the Network and Registry pages inside Vorliq.

How to Join

1. Download the Vorliq application from GitHub and follow the setup instructions in the README.

2. Run start.bat to launch the application and open your browser to localhost 3000.

3. Create your wallet, set your password, and start participating in your community economy.

The VLQ Coin

VLQ is the native coin of the Vorliq network. It is used for transfers, balances, mining rewards, community lending, and future governance activity. The total maximum supply is 21 million VLQ, and miners earn an initial reward of 50 VLQ for each block they mine.
