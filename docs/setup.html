How to Run Vorliq

Set up the complete Version 1.0 platform on Windows.

What You Need

To run a Vorliq node on Windows, you need Windows 10 or later, Node.js LTS from nodejs.org, Python 3.12 from python.org with the Add to PATH checkbox selected during installation, and Git from git-scm.com. These tools let your computer run the blockchain API, backend API, React web app, and mobile development tools.

Download Vorliq

Open a terminal in the folder where you want to install Vorliq, then clone the public repository and enter the project folder.

git clone https://github.com/vorliq/Vorliq.git
cd Vorliq

Install Dependencies

Install the blockchain dependencies first by creating the Python virtual environment, activating it, and installing the required Python packages.

cd blockchain
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

Then install the backend, frontend, and mobile JavaScript dependencies from their folders.

cd ..\backend
npm install
cd ..\frontend
npm install
cd ..\mobile
npm install

Start Vorliq

After dependencies are installed, double click start.bat in the root Vorliq folder. The startup script opens the blockchain API, backend API, React frontend, and heartbeat service. When the services are ready, open http://localhost:3000 in your browser.

Use the Platform

The web app lets you create an encrypted wallet, send VLQ, mine blocks, inspect the blockchain, join peers, register your node, request loans, vote on loans, post exchange offers, accept trades, create governance proposals, vote on proposals, monitor health, and view community statistics.

When you create a wallet, save the private key or export an encrypted wallet backup before storing meaningful VLQ. Vorliq cannot recover a lost private key or wallet password. Read the Wallet Safety guide before using a wallet on a shared or public device.

Mobile App

The Vorliq mobile app is built with React Native and Expo. Install Expo Go from the App Store or Google Play, open a terminal inside the mobile folder, run npx expo start, and scan the QR code with Expo Go. In the mobile Settings screen, set the node URL to your running Vorliq backend, usually something like http://192.168.1.20:5000 when your phone and computer are on the same network.

Join the Network

To connect to other Vorliq nodes, open the Registry page in the app, find active nodes, and click Add to My Network. Register your own node in the Registry so other community members can find it. The Network page lets you add peers directly, sync chains, and monitor peer status.

Get Help

Vorliq is a community project, so the best way to get help is to join the public channels, ask questions, and share what you are building with VLQ.
