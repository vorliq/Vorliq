Run Your Vorliq Node

This guide explains how to install Vorliq, start your local node, and connect with the community network.

What You Need

To run a Vorliq node, you need a Windows 10 or later computer, Node.js LTS from nodejs.org, Python 3.12 from python.org, and Git from git-scm.com. When installing Python, make sure the Add to PATH checkbox is selected during setup so the terminal can run Python commands correctly.

Download Vorliq

Open a terminal and navigate to the folder where you want to install Vorliq. Run git clone https://github.com/vorliq/Vorliq.git to download the project, then run cd Vorliq to move into the project folder. If your clone creates another nested Vorliq folder, open that folder before continuing.

Install Dependencies

Inside the Vorliq folder, open a terminal and go into the blockchain folder. Create the Python virtual environment by running python -m venv .venv, activate it by running .venv\Scripts\activate, and then install the Python packages with pip install -r requirements.txt. After that, go into the backend folder and run npm install. Finally, go into the frontend folder and run npm install.

Start Vorliq

To start the full application, double click start.bat in the root folder. Wait for the terminal windows to open and for the services to finish starting, then open your browser to http://localhost:3000. The browser app is where you create a wallet, mine blocks, send VLQ, view the chain, and join the network.

Join the Network

To connect to other Vorliq nodes, open the Registry page in the app, find active nodes listed there, and click Add to My Network next to any node you want to connect to. You should also register your own node in the Registry so other community members can find you and connect to your node.

Get Help

Vorliq is a community project, so the best way to keep moving is to join the public channels, ask questions, and share what you are building with VLQ.
