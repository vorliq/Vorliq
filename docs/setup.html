How to Run Vorliq

Set up the local Version 1.0 developer stack on Windows, Linux, or macOS.

What You Need

You need Node.js LTS, Python 3.12, and Git. On Windows, use Windows 10 or later and select the Python installer option that adds Python to PATH. These tools let your computer run the blockchain API, backend API, and React web app locally.

Download Vorliq

Open a terminal in the folder where you want to install Vorliq, then clone the public repository and enter the project folder.

git clone https://github.com/vorliq/Vorliq.git
cd Vorliq

Install Dependencies

Install the blockchain dependencies first by creating the Python virtual environment and installing the required Python packages.

Windows:

cd blockchain
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

Linux/macOS:

cd blockchain
python3 -m venv .venv
.venv/bin/python -m pip install -r requirements.txt

Then install the backend and frontend JavaScript dependencies from their folders.

Windows:

cd ..\backend
npm install
cd ..\frontend
npm install

Linux/macOS:

cd ../backend
npm install
cd ../frontend
npm install
cd ..
chmod +x scripts/*.sh stress_test/run_stress_test.sh

Start Vorliq

On Windows, double click start.bat in the root Vorliq folder. It opens the blockchain API, backend API, React frontend, and heartbeat helper in separate windows.

On Linux or macOS, run ./scripts/start-all.sh from the root Vorliq folder. It starts the blockchain API, backend API, and React frontend in one terminal. Press Ctrl+C to stop them. The registry heartbeat helper is disabled by default because it writes local registry heartbeat data; explicitly set VORLIQ_ENABLE_HEARTBEAT=1 when that local behavior is needed.

When the services are ready, open http://localhost:3000 in your browser. These scripts are for local development, not production deployment.

The shell scripts improve Linux/macOS developer setup coverage, but they do not claim that every operating-system configuration has been tested.

Use the Platform

The web app lets you create an encrypted wallet, send VLQ, mine blocks, inspect the blockchain, join peers, register your node, request loans, vote on loans, post peer community requests, accept coordination records, create governance proposals, vote on proposals, monitor health, and view community statistics.

When you create a wallet, save the private key or export an encrypted wallet backup before storing meaningful VLQ. Vorliq cannot recover a lost private key or wallet password. Before sending, verify the receiver address, review the amount, and remember that accepted transactions are pending until mined and cannot be reversed. Read the Wallet Safety guide before using a wallet on a shared or public device.

Mobile App

The Vorliq mobile app is built with React Native and Expo for community testing. Install Expo Go, open a terminal inside the mobile folder, run npx expo start, and scan the QR code with Expo Go. The default mobile node is https://vorliq.org; in Settings you can switch to your own backend using a safe node URL such as https://community-node.example.

The mobile app now includes wallet creation, local signing, reviewed sending, mining status, faucet claims, transaction and block detail views, profiles, lending lifecycle views and repayment, community coordination records, governance rule-change views, treasury ledger reading, treasury proposal submission and voting, node registry status, and notifications.

Some local Expo exports may warn if a private Android google-services.json file is not present. That file is only needed for builds that use a configured push-notification project. Do not commit private mobile service files or secrets.

Join the Network

To connect to other Vorliq nodes, open the Registry page in the app, find active nodes, review sync status, reliability, uptime, region, and health history, then click Add to My Network. Register your own node in the Registry so other community members can find it. Public registry fields are metadata only and should never include secrets.

Get Help

Never paste private keys, wallet passwords, admin tokens, or seed phrases into support messages, issues, public logs, or screenshots.

Vorliq is a community project, so the best way to get help is to join the public channels, ask questions, and share what you are building with VLQ.
