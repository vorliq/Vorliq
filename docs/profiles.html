Profiles and Reputation

What profiles are

Vorliq profiles are public community profiles linked to wallet addresses. They help members see a display name, avatar style, bio, public links, badges, and reputation instead of only a raw wallet address.

How profiles are linked

Each profile is keyed by one Vorliq wallet address. A profile does not prove a legal identity and should not be treated as a verified real-world identity.

What is public

If a user creates a profile, the wallet address, display name, bio, location, country, avatar style, website, X link, Telegram link, Discord name, reputation score, ambassador flag, badges, and activity summary are public.

What is not stored

Vorliq profiles must not store private keys, wallet passwords, email passwords, admin tokens, recovery phrases, or secrets. Do not paste sensitive information into profile fields.

Reputation calculation

  • +10 for having a profile
  • +5 for each earned achievement
  • +2 for each forum post
  • +1 for each forum reply
  • +3 for each completed exchange trade involving the address
  • +3 for each repaid loan
  • +2 for each governance vote
  • +2 for each treasury vote
  • +1 for each mined block

The display score is capped at 1000. Reputation is a transparent community activity score, not proof of trustworthiness, creditworthiness, identity, or financial reliability.

How to edit

Open the Vorliq app, log in with your wallet, then visit Profiles or Account and choose Create Profile or Edit Profile.

Safety reminder

Profiles are public. Do not post private information, addresses where you live, private contact details, documents, seed phrases, passwords, or anything you would not want copied outside Vorliq.
