Release History

Vorliq Release Notes

Public release notes summarize user-facing changes, compatibility status, and safe upgrade context for the Vorliq community platform.

Current Public Release

Version: 1.0.0 stable

API: v1 stable. Unversioned /api routes remain supported while /api/v1 is the recommended developer path.

Compatibility: Existing blockchain history and public API routes are preserved. Vorliq remains community blockchain software and does not provide legal, banking, investment, custody, or guaranteed financial outcomes.

Recent Production Layers

  • API versioning and developer compatibility: request IDs, API v1 aliases, stable error metadata, SDK v1 defaults, and compatibility docs.
  • Community identity verification and moderation: wallet-control profile verification, factual non-KYC trust labels, reports, and protected moderation queues.
  • Wallet and transaction safety v2: send review, address validation, duplicate-send protection, pending status clarity, and backup education.
  • Public audit exports and verification: machine-readable audit exports, manifest checks, and verification tooling for public transparency.
